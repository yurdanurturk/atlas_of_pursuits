# atlas_of_pursuits

Purpose: Suggests hobbies (pursuits) that best match a user’s Big Five personality traits, scoring each match from 0 to 100.

How it works: The user enters their 30 facet scores (1–5). The program compares these against built-in pursuits and outputs a ranked list from best to least fit, showing the score for each.

Current pursuits: 1. Arts & Creativity 2. Entertainment & Media

Features: Easy to add new pursuits by updating the array. Uses a clear, normalized scoring system. Helps users discover real-world activities that suit their personality.

Future improvements: Add more pursuits and categories. Introduce weighted facets for more accurate matching. Develop a simple web or mobile interface.
